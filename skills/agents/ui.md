# UI Preview

UI 컴포넌트를 ASCII 아트로 미리보기합니다.

## 사용법

UI 설명을 제공하면 ASCII 아트로 레이아웃을 시각화합니다.

## 지원 레이아웃

1. **sidebar** - 사이드바 + 메인 콘텐츠
2. **header-footer** - 헤더/푸터 레이아웃
3. **grid** - 그리드 레이아웃
4. **centered** - 중앙 정렬
5. **split** - 좌우 분할
6. **custom** - 커스텀 레이아웃

## 컴포넌트 타입

- header, sidebar, footer
- button, input, card
- table, list, form
- modal, dropdown, tabs

## 예시

입력: "로그인 페이지 - 이메일, 비밀번호 입력, 로그인 버튼"

출력:
```
+------------------------------------------+
|               로그인 페이지               |
+------------------------------------------+
|                                          |
|   +----------------------------------+   |
|   |  이메일                          |   |
|   +----------------------------------+   |
|                                          |
|   +----------------------------------+   |
|   |  비밀번호                        |   |
|   +----------------------------------+   |
|                                          |
|   +----------------------------------+   |
|   |           로그인 버튼            |   |
|   +----------------------------------+   |
|                                          |
+------------------------------------------+
```

## 반응형 미리보기

`responsive: true` 옵션으로 모바일 뷰도 함께 표시합니다.

## MCP 도구 연동

hi-ai의 `preview_ui_ascii` 도구를 사용합니다.
